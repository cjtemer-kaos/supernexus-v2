# Changelog - SuperNEXUS

## [2.0.0] - 2026-05-18

### Nuevo
- **DirectorNexus v2**: Cerebro central con routing inteligente de tareas
- **22 Gemas especializadas**: Cada una con herramientas y modelos propios
- **Interfaz web moderna**: React 19 + Vite + TypeScript
- **Chat con streaming WebSocket**: Respuestas token por token en tiempo real
- **Vision por computadora**: Analisis de imagenes con modelos locales
- **Control de PC**: Capturas, mouse, teclado automatizados
- **CodeGraph**: Motor semantico de busqueda de codigo (AST)
- **Busqueda FTS5**: Busqueda de texto ultra rapida
- **Skill Hub**: 1,400+ habilidades indexadas
- **API compatible con OpenAI**: Para usar con OpenWebUI y otros clientes
- **Auth integrada**: Setup de cuenta en primer uso
- **Sistema de migracion v1->v2**: Preserva todos tus datos

### Mejorado
- Arquitectura modular con extension host
- Lazy loading de gemas (menor uso de memoria)
- Seguridad mejorada con guardrails y rate limiting
- Optimizacion de tokens automatica

### Cambios
- Interfaz migrada de Electron a web nativa
- Base de datos unificada para memoria
- Rutas de configuracion estandarizadas
